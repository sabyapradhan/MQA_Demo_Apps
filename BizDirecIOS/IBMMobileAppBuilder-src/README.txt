Requeriments
------------

- XCode 7
- Cocoapod 0.39.0

Before starting
---------------
1.- Install cocoapods (if needed) - http://guides.cocoapods.org/using/getting-started.html
    * sudo gem update --system
    * sudo gem install cocoapods
    * pod setup

2.- Create workspace
    * pod install

3.- Open workspace and enjoy  

Update IBMMobileAppBuilder library (debug)
------------------------------------------

1.- Open workspace

2.- Apply changes

3.- Select BuildLibraryDebug target

4.- Run target

5.- Link files in app project

Update IBMMobileAppBuilder library (release)
--------------------------------------------

1.- Open workspace

2.- Apply changes

3.- Select BuildLibrary target

4.- Run target

5.- Link files in app project

Create and add IBMMobileAppBuilder library
------------------------------------------

1.- Open workspace

2.- Apply changes (optional)

3.- Select BuildLibraryDebug target

4.- Run target

5.- Open your app project

6.- Remove reference to debug library (in group IOSApp/IBMMobileAppBuilder/)

7.- Add generated library to group IOSApp/IBMMobileAppBuilder/
